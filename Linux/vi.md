## vi /vim 편집기 명령어

## vi 모드

#### 명령모드

- vi를 실행시킨 후 기본 모드. 커서의 이동,수정,삭제,복사,탐색 등의 역할
- 입력 모드 전환키인 `i, a, o` 를 입력하면 입력모드로 전환

#### 입력모드

- 글자를 입력하는 문서를 만드는 모드
- 화면 아래에 `--INSERT--` 라고 표시됨
- 명령모드로 전환하려면 `esc`

#### ex 명령 모드

- 명령 모드에서 `:` 키를 입력했을때 맨 아랫줄에서 명령 수행하는 모드
- 저장, 종료, 탐색, 치환 및 vi 환경 설정 역할

<br>

## 파일 만들기, 편집하기

- `vi 파일명` : 파일명의 파일이 존재하면 그 파일을 열고, 존재하지 않으면 새 파일 생성

## 저장, 종료

- `:q` : + 엔터 vi 편집기 종료
- `:w` : 저장
- `:q!` : 변경된 내용이 있어도 저장하지 않고 종료
- `:wq` : 변경된 내용을 저장하고 종료

<br>

## 커서 이동

- `h, j, k, l` : 좌 하 상 우
- `w` : 다음 단어의 처음
- `gg` : 제일 첫 행으로 이동
- `G` : 제일 끝행으로 이동 (Shift + g)

## 삭제

- `dd` : 현재 커서 행 삭제
- `dw` : 단어 삭제

## 복사, 붙여넣기

- `yy` : 현재 커서의 행 복사
- `p` : 복사한 내용을 현재 행 이후에 붙여넣기
