## 변수와 입력

### 변수만들기

- x = 10 이라고 입력하면 10이 들어있는 변수 x가 만들어짐
- 즉, `변수이름 = 값` 형식
- 영문은 대소문자를 구분하며 숫자를 사용할 수 있다.
- 문자부터 시작해야하며 숫자부터 시작할 수 없다.
- `_(밑줄문자)`로 시작할 수 있다.
- 특수문자`(+, -, *, /, $, @, & 등)`는 사용할 수 없다.
- 파이썬의 키워드`(if, for, while, and, or 등)`는 사용할 수 없다.

```py
>>> x = 10
>>> x
10
```

### 변수의 자료형 알아내기

- 파이썬에서는 변수의 자료형이 중요하다.
- `type`에 변수를 넣으면 변수(객체)의 자료형이 나온다.
- 변수를 사용하다보면 자료형이 맞지않는 문제를 자주접하는데, `type`으로 자료형이 뭔지 알아보면 쉽게 해결가능하다.

```py
>>> y = 'Hello'
>>> type(y)
<class 'str'>
```

### 변수 여러 개를 한 번에 만들기

- 변수 여러 개를 한 번에 만들 수도 있다.
- 변수와 값의 개수는 동일하게 맞춰줘야 한다.

```py
>>>x,y,z=10,20,30
>>> x
10
>>> y
20
>>> z
30
```

- 변수 여러 개를 만들때 값이 모두 같아도 된다면 다음과 같은 방식도 가능하다.

```py
>>> x = y = z = 10
>>> x
10
>>> y
10
>>> z
10
```

- 두 변수의 값을 바꿀 수 있다. 할당할 때 서로 자리를 바꾸면 된다.

```py
>>> x, y = 10, 20
>>> x, y = y, x
>>> x
20
>>> y
10
```

### 변수 삭제하기

> 변수 삭제는 `del`을 사용

```py
>>> x = 10
>>> del x
>>> x
# Traceback (most recent call last):
#   File "<pyshell#31>", line 1, in <module>
#     x
# NameError: name 'x' is not defined
```

### 빈 변수 만들기

- 값이 들어있지 않은 빈 변수를 만들때는 `None`을 할당한다.
- 보통 다른 언어에서는 널`(null)`이라고 표현한다.

```py
>>> x = None
>>> print(x)
None
>>> x
# 아무것도 출력되지 않음
```

### 변수로 계산하기

> 변수는 변수끼리 계산가능하고, 계산 결과를 다른 변수에 할당할 수 있다.

```py
>>> a = 10
>>> b = 20
>>> c = a + b
>>> c
30
```

### 산술 연산 후 할당 연산자 사용

변수 a값에 20을 더하면 30이 나오지만 a를 다시 출력해보면 10이 나온다.

```py
>>> a = 10
>>> a + 20
30
>>> a
10
```

- `a + 20`은 `a`에 `20`을 더하기만하고 계산 결과를 유지하지 않는다.
- 변수에서 값의 변화를 계속 유지하려면 계산 결과를 다시 변수에 저장한다.

```py
>>> a = 10
>>> a = a + 20
>>> a
30
```

- 변수를 두 번 입력하지 않고 산술 연산후 할당 연산자를 사용할 수 있다.

```py
>>> a = 10
>>> a += 20
>>> a
30
```

- 뺄셈`(-=)`, 곱셉`(*=)`, 나눗셈`(/=, //=)`, 나머지`(%=)`도 같은 방식이다.

### 입력 값을 변수에 저장하기

#### input 함수 사용

```py
>>> input()
Hello World		# 입력한 값
'Hello World'
```

입력한 문자열이 그대로 출력된다. `input`은 사용자가 입력한 값을 가져오는 함수이다.

#### input 함수 결과를 변수에 할당하기

`변수 = input()` 으로 결과를 할당할 수 있다.

```py
>>> x = input()
Hello World		# 입력한 값
>>> x
'Hello World'
```

input괄호 안에 문자열을 지정하면 입력받는 상태인지 출력이없는 상태인지 알 수 있다.

```py
>>> x = input('문자열을 입력하세요: ')
문자열을 입력하세요: Hello World
>>> x
'Hello World'
```

#### 두 숫자의 합 구하기

```py
>>> a = input('첫 번째 숫자를 입력하세요: ')
>>> b = input('두 번째 숫자를 입력하세요: ')
>>> print(a + b)
```

숫자를 입력하면 결과가 문자열 형태이기 때문에 그대로 붙어서 나온다.

```py
>>> a = input('문자열을 입력하세요: ')
문자열을 입력하세요: 10
>>> b = input('문자열을 입력하세요: ')
문자열을 입력하세요: 20
>>> print(a + b)
1020
```

```py
>>> type(a)
<class 'str'>
```

`input`으로 입력받은 것은 항상 문자이기 때문이다.

### 입력 값을 변수 두 개에 저장하기

- input에서 split을 사용한 변수 여러 개에 저장한다. 각 변수는 콤마로 구분해준다.
- input에 split을 사용하면 입력받은 값을 공백을 기준으로 분리하여 변수에 차례로 저장한다.

```py
>>> a, b = input('문자열 두 개를 입력하세요: ').split()
문자열 두 개를 입력하세요: apple mint
>>> print(a)
apple
>>> print(b)
mint
```

마찬가지로 숫자 두개를 더하려면 연결된 결과가 출력되기 때문에 변수를 정수로 변환해야 한다.

```py
>>> a, b = input('문자열 두 개를 입력하세요: ').split()
문자열 두 개를 입력하세요: 10 20
>>> a = int(a)
>>> b = int(b)
>>> print(a + b)
30
```

`print`안에서 `int`로 변수를 변환하고 바로 더해도 된다.

```py
print(int(a) + int(b))
```

#### map을 사용하여 정수로 변환하기

- `map`에 `int`와 `input().split()`을 넣으면 `split`의 결과를 모두 `int`로 변환해준다. (실수로 변환할 때는 `float`)

```py
>>> a, b = map(int, input('숫자를 입력하세요: ').split())
숫자를 입력하세요: 11 22
>>> print (a + b)
33
```

#### 입력받은 값을 콤마를 기준으로 분리하기

> split에 기준 문자열을 지정하여 공백이 아닌 다른 문자로 분리하기

```py
>>> a, b = map(int, input('숫자를 입력하세요: ').split(','))
숫자를 입력하세요: 1,2
>>> print(a + b)
3
```

이렇게 `split`의 결과를 모두 `int, float`로 변환할 때는 `map`을 사용하면 편리하다.
