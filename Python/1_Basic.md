## 알고리즘과 코딩

문제의 패턴 발견과 해결 절차

- 문제에서 일정한 패턴을 발견하고, 패턴을 토대로 문제를 해결하는 절차가 알고리즘
- 알고리즘을 코드로 표현하는 행동을 코딩(알고리즘을 컴퓨터 명령으로 작성하는 것을 프로그래밍이라고 하며 코딩과 같은 개념)

프로그래밍과 코딩?

- 둘 다 같은 작업을 지칭함
- 프로그래밍은 컴퓨터 명령어로 표현한다는 뜻
- 코딩은 파이썬 등의 프로그래밍 언어로 코드를 작성한다는 뜻

## 세미콜론

- 파이썬은 세미콜론을 붙이지 않는다.
- 세미콜론을 붙여도 문법 에러는 발생하지 않음

```py
print('Hello, World!')
```

- 보통 한 줄에 여러 구문을 사용할 때 세미콜론으로 구분

```py
print('Hello'); print('1234')
```

## 주석(Comment)

- 파이썬에서 사람만 알아볼 수 있도록 작성한 부분
- 주석은 인터프리터가 처리하지 않으므로 프로그램의 실행에는 영향을 주지않음
- 한줄 주석과 블록 주석 두 가지가 있음

### 한줄 주석

```py
# Hello, World 출력
print('Hello World')
```

코드 뒤에 `#`으로 주석을 작성할 수도 있다.

```py
print('Hello World') # Hello, World 출력
```

### 블록 주석

> 각 줄마다 맨 앞에 `#`을 넣어준다. 읽기 쉽게 `#`뒤에 공백을 한칸 띄운다.

```py
# 블록
# 주석
```

### 들여쓰기

- 파이썬은 들여쓰기 자체가 문법이다.
- 들여쓰기를 하지 않으면 문법 에러가 발생한다.

```py
if a == 10;
	print('10입니다')
```

`공백 2칸, 공백4칸, Tab키`등을 사용하여 들여쓰기 한다.

### 코드 블록

- 코드 블록은 특정한 동작을 위해 코드가 모여있는 상태
- 파이썬은 들여쓰기를 기준으로 코드 블록을 구성. 같은 블록은 들여쓰기 칸 수가 같아야 한다.

```py
if a == 10;
	print('10')
	print('입니다')
```

## 숫자 계산하기

- 파이썬은 숫자를 정수`(int)`, 실수`(float)`, 복소수`(complex)`로 구분한다.

### 사칙연산

- `파이썬3`는 나눗셈이 완전히 나누어 떨어져도 실수가 나옴

```py
>>> 4 / 2
2.0
```

- 나눗셈후 소수점 이하를 버리는 `//` 연산자

```py
 >>> 5//2
 2
 >>> 4//2
 2
 >>> 5.5//2
 2.0
```

- 나눗셈 후 나머지를 구하는 `%` 연산자

```py
>>> 5%2
1
```

- 거듭제곱을 구하는 `**` 연산자

```py
>>> 2**3
8
```

### 값을 정수로 만들기

- `int`에 괄호를 붙이고 숫자 또는 계산식을 넣는다
- 문자열을 넣어도 정수로 만들 수 있다. 단, **정수로 된 문자열**이어야 한다.

```py
>>> int(3.3)
3
>>> int('10')
10
>>> int('3.3')
# ValueError: invalid literal for int() with base 10: '3.3'
```

### 객체의 자료형 알아내기

```py
>>> type(10)
<class 'int'>
```

- `type`은 객체의 타입(자료형)을 알아내는 함수
- `<class 'int'>`는 정수`(int)`클래스라는 뜻
- 파이썬에서는 숫자도 객체`(object)`이며 객체는 클래스`(class)`로 표현

### 몫과 나머지를 함께 구하기

- 몫과 나머지를 함께 구하려면 `divmod`를 사용

```py
>>> divmod(5, 2)
(2, 1)
```

- 파이썬에서 값을 괄호로 묶은 형태를 `튜플(tuple)`이라고 하고 값 여러개를 모아서 표현할 때 사용

```py
>>> a, b = divmod(5,2)
>>> print(a,b)
2 1
```

- 튜플은 변수 여러 개에 저장할 수 있는데 `divmod`의 결과가 튜플로 나오므로 몫과 나머지는 변수 두 개에 저장할 수 있음

### 2진수, 8진수, 16진수

- 정수는 10진수 외에도 2,8,16진수로 표현가능
- 2진수 : 숫자앞에 0b를 붙이며 0과 1사용
- 8진수 : 숫자앞에 0o를 붙이며 0부터 7까지 사용
- 16진수 : 숫앞에 0x를 붙이며 0부터 9, A부터 F까지 사용

```py
>>> 0b110
6
>>> 0o10
8
>>> 0xf
15
```

## 실수 계산하기

```py
>>> 4.3-2.7
1.5999999999999996
```

`1.6`이아니라 .99999가 나온이유는 오차라는것이 발생하기 때문이다.

### 실수와 정수 함께 계산

```py
>>> 4.2 + 5
9.2
```

- 실수와 정수를 함께 계산하면 **표현범위**가 넓은 실수로 계산된다.

### 값을 실수로 만들기

- `float`에 괄호를 붙이고 숫자 또는 계산식을 넣는다.
- 문자열을 넣어도 되지만 정수로된 문자열이어야 한다.

```py
 >>> float(5)
5.0
>>> float( 1+ 2)
3.0
>>> float('5.4')
5.4
```

## 괄호 사용하기

- 연산의 우선순위는 파이썬에서도 곱셈보다 뎃셈이 먼저다.

```py
>>> 35 + 1 * 2
37
```

- 덧셈을 먼저 계산하려면 그부분을 괄호로 묶으면 된다.

```py
>>> (35 + 1) *2
72
```
