# 리스트 응용(1) - 리스트 조작하기

> 파이썬의 리스트는 생각보다 기능이 많다. 요소를 추가/삭제하거나, 정보를 조회하는 메서드(함수)도 제공한다.

## 리스트에 요소 추가하기

- `append` : 요소 하나를 추가
- `extend` : 리스트를 연결하여 확장
- `insert` : 특정 인덱스에 요소 추가

### 리스트에 요소 하나 추가하기

```py
>>> a = [10, 20, 30]
>>> a.append(555)
>>> a
[10, 20, 30, 555]

>>> len(a)
4
```

물론 빈 리스트에 값을 추가할 수도 있다.

### 리스트 안에 리스트 추가하기

```py
>>> a = [1, 2, 3]
>>> a.append([555,666])
>>> a
[1, 2, 3, [555, 666]]

>>> len(a)
4
```

### 리스트 확장하기

> extend는 리스트 끝에 다른 리스트를 연결하여 확장한다.

```py
>>> a = [10,20,30]
>>> a.extend([444,999])
>>> a
[10, 20, 30, 444, 999]

>>> len(a)
5
```

### 리스트의 특정 인덱스에 요소 추가하기

- `insert(인덱스, 요소)`는 리스트의 특정 인덱스에 요소 하나를 추가한다.

```py
>>> a = [11,22,33]
>>> a.insert(2, 9999)
>>> a
[11, 22, 9999, 33]
```

- 리스트 마지막에 요소 추가 : `insert(len(리스트), 요소)`
- 리스트 맨 처음에 요소 추가 : `insert(0, 요소)`
  <br>
- `insert`는 요소 하나를 추가하므로 `insert`에 리스트를 넣으면 `append`처럼 리스트 안에 리스트가 들어간다.

```py
>>> a = [10, 20, 30]
>>> a.insert(1, [500,600])
>>> a
[10, [500, 600], 20, 30]
```

```py
>>> a = [10, 20, 30]
>>> a[1:1] = [500, 600]
>>> a
[10, 500, 600, 20, 30]
```

## 리스트에서 요소 삭제하기

- `pop` : 마지막요소 또는 특정 인덱스의 요소를 삭제
- `remove` : 특정 값을 찾아서 삭제

### 리스트에서 특정 인덱스 요소를 삭제하기

- `pop()`은 리스트 마지막요소를 삭제하고 삭제한 요소를 반환한다.

```py
>>> a = [10,20,30]
>>> a.pop()
30

>>> a
[10, 20]
```

<br>

- `pop(인덱스)`는 해당 인덱스 요소를 삭제한 뒤 삭제한 요소를 반환한다.

```py
>>> a = [10, 20, 30]
>>> a.pop(1)
20

>>> a
[10, 30]
```

<br>

- `pop`대신 `del`을 사용해도 상관없다.

```py
>>> a = [10,20,30]
>>> del a[1]
>>> a
[10, 30]
```

### 리스트에서 특정 값을 찾아서 삭제하기

- `remove(값)`은 리스트에서 특정 값을 찾아 삭제한다.

```py
>>> a = [10,20,30]
>>> a.remove(20)
>>> a
[10, 30]
```

- 리스트에 해당 값이 여러개면 처음 값을 삭제한다.

```py
>>> a = [10,20,40,20]
>>> a.remove(20)
>>> a
[10, 40, 20]
```

들어갈때 ABC 순으로 쌓으면 나갈때 CBA로 나간다 ( `appen()`는 추가 , `pop()`는 제거 )

#### 리스트로 큐 만들기

들어오는곳과 나가는곳 다름. ABC 순으로 들어가면 ABC로 나간다

## 리스트에서 특정 값의 인덱스 구하기

- `index(값)`은 리스트에서 특정 값의 인덱스를 구한다.
- 여러개일 경우 처음 찾은 인덱스를 구한다.

```py
>>> a = [10,20,30,20,15]
>>> a.index(20)
1
```

## 특정 값의 개수 구하기

- `count(값)`은 리스트에서 특정 값의 개수를 구한다.

```py
>>> a = [10,20,30,15,20,20]
>>> a.count(20)
3
```

## 리스트의 순서를 뒤집기

- `reverse()`는 리스트에서 요소의 순서를 반대로 뒤집는다.

```py
>>> a = [10,20,30,15,20,20]
>>> a.reverse()
>>> a
[20, 20, 15, 30, 20, 10]
```

## 리스트의 요소를 정렬하기

- `sort()`는 리스트의 요소를 작은 순서대로 정렬함(오름차순)

```py
>>> a = [10,20,55,14,3]
>>> a.sort()
>>> a
[3, 10, 14, 20, 55]
```

- `sort(reverse=True)`는 리스트의 값을 큰 순서대로 정렬함(내림차순)

## 리스트의 모든 요소를 삭제하기

- `clear()`는 리스트의 모든 요소를 삭제한다.

```py
>>> a = [10,20,30]
>>> a.clear()
>>> a
[]
```

- `del a[:]`와 같이 시작/끝 인덱스를 생략해서 모든 요소를 삭제할 수도 있다.

```py
>>> a = [10,20,30]
>>> del a[:]
>>> a
[]
```

## 리스트를 슬라이스로 조작하기

- 리스트 끝에 값이 한 개 들어있는 리스트를 추가하기

```py
>>> a = [10,20,30]
>>> a[len(a):] = [500]
>>> a
[10, 20, 30, 500]
```

`a.append(500)`과 같다.

## 리스트의 할당과 복사

`b = a`와 같이 리스트를 다른 변수에 할당하면 리스트는 두 개가 될것 같지만 실제로는 한 개다.

```py
>>> a = [0,0,0,0]
>>> b = a
```

```py
>>> a is b
True        # 변수 이름만 다를뿐 리스트 a,b는 같은 객체

>>> b[2] = 99
>>> a
[0, 0, 99, 0]       # b리스트의 요소를 변경하면 a에도 적용됨
```

### 리스트 복사

- 리스트 a와 b를 완전히 두개로 만드려면 `copy`메서드로 복사한다.

```py
>>> a = [0,0,0,0]
>>> b = a.copy()
```

<br>

- a와 b를 `is` 연산자로 비교해보면 `False`가 나온다. 두 리스트는 다른 객체이다.

```py
>>> a is b
False

>>> a == b      #복사된 요소는 같으므로 ==로 비교하면 True
True

>>> b[2] = 99   #리스트 a와 b는 별개이므로 한쪽을 변경해도 다른 리스트에 영향X
>>> a
[0, 0, 0, 0]
>>> b
[0, 0, 99, 0]
```

## 반복문으로 리스트 요소 모두 출력하기

### for 반복문으로 요소 출력하기

```
for 변수 in 리스트:
    반복할 코드
```

```py
>>> a = [12, 44, 77, 33]
>>> for i in a:
	    print(i)

# 12
# 44
# 77
# 33
```

<br>

### 인덱스와 요소를 함께 출력하기

- `for 인덱스, 요소 in enumerate(리스트):`

```py
>>> for index, value in enumerate(a):
	    print(index, value)

# 0 12
# 1 44
# 2 77
# 3 33
```

### 인덱스 1부터 출력하기

```py
>>> for i, v in enumerate(a, start=1):
	    print(i, v)

# 1 12
# 2 44
# 3 77
# 4 33
```

### while 반복문으로 요소 출력하기

```py
>>> a = [12, 44, 77, 33]
>>> i = 0
>>> while i < len(a):
    	print(a[i])
    	i += 1

# 12
# 44
# 77
# 33
```

`i <= len(a)`처럼 `<=`를 사용하면 리스트 범위를 벗어나게 되므로 주의한다.

<br>

## 리스트의 가장 작은수, 큰수 구하기

- 리스트의 가장 작은 수 구하기

```py
>>> a = [38, 22, 18, 62, 44]
>>> smallest = a[0]
>>> for i in a:
    	if i < smallest:
	    	smallest = i

>>> smallest
18
```

숫자를 계속 비교해서 숫자가 작으면 `smallest`를 바꾸는 방식이다.

- 리스트의 가장 큰 수 구하기 : 위의 코드에서 **부등호를 반대로**하면 된다.

### 간단한 방법 - sort()

- 리스트를 작은 순서대로 정렬(오름차순)하면 첫 번째 요소가 가장 작은 수다.
- 반대로 큰 순서대로 정렬(내림차순)도 첫 번째 요소가 가장 큰 수다.

```py
>>> a = [38, 22, 18, 62, 44]
>>> a.sort()
>>> a[0]
18
>>> a.sort(reverse=True)
>>> a[0]
62
```

### 더 간단한 방법 - min, max

- 파이썬에서 제공하는 `min, max`함수를 사용하면 된다.

```py
>>> a = [38, 22, 18, 62, 44]
>>> min(a)
18
>>> max(a)
62
```

## 리스트의 합계 구하기

- 요소의 합계 구하기
- 합계를 구할 때도 반복문을 사용할 수 있다.

```py
>>> a = [1,2,3,4,5]
>>> x = 0
>>> for i in a:
    	x += i


>>> x
15
```

- `sum`함수를 사용하면 더 간단하게 합할 수 있다.

```py
>>> a = [1,2,3,4,5]
>>> sum(a)
15
```
