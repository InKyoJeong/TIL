# if 조건문 사용하기

- if 에 조건식을 지정하고 `:`을 붙이며 다음줄에 실행할 코드가 온다.
- 실행할 코드는 반드시 들여쓰기 해야한다.

```py
>>> x = 10
>>> if x == 10:
	print('Hello')


# Hello
```

## if 조건문에서 코드를 생략하기

- `pass`는 아무일도 하지않고 넘어간다.
- `pass`만 넣고 나중에 할일은 주석으로 남겨놓는 방식이다.

```py
>>> x = 5
>>> if x == 5:
	pass            # to-do: x가 5일때

>>>
```

## if 조건문과 들여쓰기

```py
x = 5

if x == 10:
    print('x는')
print('10이다.')

# 10이다
```

## 중첩 if 조건문 사용하기

변수 값이 10이상이면 10이상이다, 15이면 15이다, 20이면 20이다를 출력하는 예제

```py
>>> x = 15
>>> if x >= 10:
	    print('10 이상이다.')

        if x == 15:
            print('15이다.')

        if x == 20:
            print('20이다.')

# 10 이상이다.
# 15 이다.
```

## 사용자가 입력한 값에 if 조건문 사용하기

```py
x = int(input())

if x == 10:
    print('10입니다.')
```

## else 사용하기

```py
>>> y = 4
>>> if y == 10:
	    print('y는 10이다.')
    else:
	    print('10이 아니다.')

# 10이 아니다.
```

```py
>>> if True:
	    print('참')
    else:
	    print('거짓')

# 참
```

- `None`은 `False`로 취급되므로 `else`의 코드가 실행된다.

```py
>>> if None:
	    print('참')
    else:
	    print('거짓')

# 거짓
```

<br>

- 숫자는 정수,실수와 관계없이 `0`이면 거짓, `0`이 아니면 참이다.

```py
>>> if 0:
	    print('참')
    else:
    	print('거짓')

# 거짓

>>> if 1234:
	    print('참')
    else:
	    print('거짓')

# 참
```

<br>

- 빈 문자열은 거짓, 문자열은 내용이 있으면 참이다.

```py
>>> if 'Hello':
	    print('참')
    else:
	    print('거짓')

# 참

>>> if'':
	    print('참')
    else:
    	print('거짓')

# 거짓
```

## 조건식을 여러 개 지정하기

```py
>>> x = 10
>>> y = 20

>>> if x == 10 and y == 20:
    	print('참')
    else:
    	print('거짓')

# 참
```

## elif를 이용하여 여러방향으로 분기하기

- `if,else`와 마찬가지로 조건식 끝에 `:`(콜론)을 붙여야하고, `elif`단독으로 사용할 수 없다.

```py
if 조건식:
    코드1
elif 조건식2:
    코드2
else:
    코드3
```
